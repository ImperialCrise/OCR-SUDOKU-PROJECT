# OCR-Sudoku-Solver
OCR Sudoku Solver Projet S3 Epita. Here is a [link to our website](https://www.justwood-game.com/ocr.php).

## Installation
You need to have these installed :
 * libsdl2-dev
 * libsdl2-ttf-dev
 * libgtk-3-dev

To compile our project :
```bash
make
```

## Usage
To run our project :
```bash
./ocr
```

## Clean project
To clean the project :
```bash
make clean
```
